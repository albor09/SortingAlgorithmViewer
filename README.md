# SortingAlgorithmViewer

A C# Windows application that visualizes different sorting algorithms in action.

![Sorting Algorithm Viewer Demo](https://github.com/user-attachments/assets/ae45c0b2-1282-4620-85fd-969bc6b2e5de)

## Features

- Visual step-by-step representation of sorting processes
- Multiple sorting algorithms (implemented in the source code)
- Clean, educational interface for understanding sorting mechanics
